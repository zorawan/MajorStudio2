Iconography

The floppy disk icon represented a vital action when the early user interface was introduced. Save files was once an essential action required in many software products. For designers who work on the user interface need to cooperate with icons to deliver, an explicit affordance would use floppy disk icons for saving during the 80s and 90s. According to IBM, the company that creates the floppy disk: "The icon of a floppy disk remains the universal symbol for saving files." Indeed, the floppy disk icon was recognizable by almost everyone who has ever used a personal computer. However, the icon disappears from almost all the user interfaces.
